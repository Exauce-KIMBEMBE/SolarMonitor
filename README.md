# SolarMonitor
An IoT-based solar tracking and monitoring platform using ESP32 for real-time panel positioning, environmental sensing, remote control, and cloud-connected data visualization.

The project combines an ESP32, Arduino, light sensors (LDR), temperature sensors, and a web platform to enable remote control and monitoring. The system supports automatic solar tracking based on light intensity, predefined angle positioning, real-time measurements, and remote commands through a web interface.

Main features:
• Automatic solar tracking using LDR sensors
• Manual and predefined panel positioning
• Real-time monitoring of panel angle and sensor data
• Temperature and light intensity measurements
• Remote control through a web dashboard
• User authentication and account management
• Cloud-based communication between ESP32 and the web platform
• Data visualization and system monitoring
